# 🌐 Web Portfolio - Nabiil Fanezi

Website portofolio pribadi yang dibangun menggunakan **React** dan **Tailwind CSS** untuk menampilkan profil, keterampilan, proyek, serta informasi kontak secara modern, responsif, dan interaktif.

## ✨ Fitur

- Landing page modern dan responsif
- Section Tentang Saya
- Daftar keterampilan (Skills)
- Showcase proyek yang pernah dikerjakan
- Animasi interaktif menggunakan Animate.css dan AOS
- Navigasi yang sederhana dan mudah digunakan
- Tampilan yang optimal pada desktop maupun perangkat mobile

## 🛠️ Teknologi yang Digunakan

| Teknologi | Deskripsi |
|------------|-------------|
| React | Library JavaScript untuk membangun antarmuka pengguna berbasis komponen |
| Vite | Build tool modern dengan proses development yang cepat |
| Tailwind CSS | Framework utility-first untuk styling yang fleksibel dan responsif |
| Animate.css | Library animasi siap pakai untuk berbagai efek transisi |
| AOS (Animate On Scroll) | Library animasi berbasis scroll untuk meningkatkan pengalaman pengguna |
